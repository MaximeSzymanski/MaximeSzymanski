### 🚀 Featured Projects

- 🍔 [**FastFoodSim: Digital Twin & AI Optimization**](https://github.com/MaximeSzymanski/FastFoodSim)
  A high-fidelity Discrete-Event Simulation of a fast-food kitchen. Built to test RL agents against real-world constraints (human variance, perishable inventory). Features an autonomous **Optuna** staffing optimizer and a containerized training pipeline.
  *Tech: SimPy, Stable-Baselines3 (Maskable PPO), Gymnasium, Docker, MLOps.*

- 🤖 [**Data Viz Assistant: GenAI for Analytics**](https://github.com/MaximeSzymanski/DataVizAssistant)
  Engineered an autonomous, self-healing code-generation agent capable of translating natural language queries into executable Python visualization scripts. The agent parses execution tracebacks and iteratively corrects its own code.
  *Tech: LangChain, LLMs, Pandas, Matplotlib, Streamlit.*

- 🎲 [**SplendIA: AI for Splendor Board Game**](https://github.com/MaximeSzymanski/splendia)
  Led the development of a competitive AI agent using **Self-Play Reinforcement Learning (PPO)** alongside traditional algorithms (Monte-Carlo Tree Search, Alpha-Beta pruning) to optimize decision-making without human data.

- 📈 [**The Impact of Rating Systems on Multi-Agent RL**](https://github.com/MaximeSzymanski/RatingAlgoComparison)
  Analyzed the stability of rating algorithms in a MARL environment. Demonstrated that coupling the **Glicko-2** rating system with a **DQN** agent improved population diversity by 25%, significantly reducing mode collapse during training.
